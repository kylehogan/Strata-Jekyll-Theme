Current Projects

My current research is within the Modular Approach to Cloud Security (MACS), Massachusetts Open Cloud (MOC), and PRIMES projects. MACS is a NSF Frontier project collaboration that focuses on building secure cloud services. The MOC is a public cloud in the open cloud exchange model that provides tenants with the ability to mix and match providers for different services, use federated identity between different providers, and obtain strong isolation through its bare metal sevices. PRIMES is a math and computer science enrichment program for high school students organized by MIT.

Universal Composability

Universal Composability guarantees that a protocol proved UC secure will remain secure regardless of what other protocols it is composed with or whether other instances of the protocol are running concurrently.

UC Network Time Protocol

NTP is used to proide good time to clients and is relied on for many security ritical protocols including certificate verification. This project focuses on providing a UC proof of its security that would allow it to be used as a subroutine in proofs for other protocols like TLS.

Universally Composable OpenStack

OpenStack is an open source, modular cloud computing framework. This project intends to provide a proof of security for an entire cloud by composing proofs for its components.

Bare Metal Clouds

Bare metal clouds provide users with physical machines rather than virtual ones.

Secure Cloud

This project seeks to addres the security concerns specific to bare metal clouds. We use a TPM as a hardware root of trust and measure each step of the boot process along with the machine's firmare before booting the client's image or attaching their storage. If the values in the TPM's registers do not match those on a whitelist, the machine will not finish booting and encryption keys to the client's stroage will not be releassed.

Hardware Isolation Layer

HIL is the bare metal cloud platform for the MOC. It provides network isolation for physical nodes and works with the Bare Metal Imaging project to quickly allocate and deallocate nodes. Secure Cloud is built on top of HIL.

Cloud Side Channels

Side channels become a problem whenever there is shared hardware. HIL avoids most traditional side channels by giving tenants an entrire physical machine, but some of the network infrastructure is still shared between tenants.

Network Flooding

This project seeks to exploit shared network infrastructure by using an observed attacker bandwidth to determine victim activity. Initially it focused of virtualized instances colocated on the same MOC node. These instances share a physical NIC so an attacker that is able to consume all the avialable bandwith out of the node is able to observe when a colocated victim begins sending packets through the corresponding drop in their own bandwidth.

MPC for Vulnerability Data Aggregation

A common problem organizations face is determining how to prioritize security updates and patches to minimize the risk of vulnerabilities in their infrastructure. This project uses secure multiparty computation (MPC) to allow organizations to perform joint analytics while maintaining the confidentiality of their own data. We designed and implemented MPC protocols for two computations with such privacy concerns: selecting IP addresses to blacklist and aggregating the output from vulnerability scanners. Paper Slides

About Me

My interests are primarily in applied cryptography, computer systems, network security, secure hardware, side channels, and cloud computing. I would like to develop protocols that allow sensitive tasks to be run on only semi-trusted resources. Current cloud architectures require a high level of trust in both the provider and other tenants. Reducing this trust and providing enforceable security guarantees is difficult, particularly against high privilege system components, but is necessary for providing confidentiality and integrity for outsourced tasks.

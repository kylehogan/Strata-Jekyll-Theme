Secure Multiparty Computation at Scale

Incentivizing MPC

This project focuses on providing appropriate incentives to outweigh the cost of participating in a multiparty computation. We consider how to best assess the value of participants' inputs in order to preferentially provide outputs to those participants who had the most valuable inputs. In this way, a party with high quality data could be motivated to participate in the computation through a guarantee that they would receive an output with more utility than would be provided to the other participants - e.g. they would receive their output earlier or with less noise. The report can be found here.

Cybersecurity

Analysis of Secure Storage in Android Applications

For ths project we focused on reverse engineering Android applications designed for secure password storage. We provide an analysis of to what degree these applications meet their goals as well as a discussion of the code obfuscation used in the construction of the applications. A report can be found here.
